# Exercícios de Lógica em Python

Repositório com exercícios resolvidos em Python, do básico ao avançado, com foco em lógica de programação.

## Estrutura

- Básico: variáveis, entrada/saída, condicionais
- Intermediário: loops, listas, funções
- Avançado: algoritmos, recursão, manipulação de dados
- Desafios: problemas mais complexos
- Projetos simples: aplicações práticas

## Objetivo

Desenvolver raciocínio lógico e prática em Python para o mercado de trabalho.
