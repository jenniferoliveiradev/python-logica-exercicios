# Desafios

Desafios de Programação em Python

Esta pasta contém desafios práticos de lógica e programação,
com problemas que exigem análise e raciocínio para serem resolvidos.

Conteúdo esperado:
* Desafios com condicionais e tomada de decisão
* Problemas com laços e repetição
* Manipulação de listas e estruturas de dados
* Exercícios de lógica combinatória
* Problemas que integram múltiplos conceitos

Objetivo: desenvolver pensamento crítico e habilidade de
resolver problemas reais através da programação.
